Hexo发布到GitHub常见问题

2020-04-15

hexo deploy 上传 README.md到 github

  1. 将README.md问文件移动到source目录
  2. 在hexo的_config.yml中设置skip_render: README.md

hexo deploy 每次都要自定义域名

  1. 将CNAME文件移动到source目录
  2. 将在Github Page中自定义的域名写到CNAME里
